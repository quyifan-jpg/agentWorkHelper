# AIWorkHelper Web 项目启动指南


## 项目结构
```
aiworkhelper-web/
├── public/                # 静态资源目录
├── src/
│   ├── api/              # API 接口定义
│   │   ├── user.ts       # 用户相关接口
│   │   ├── todo.ts       # 待办事项接口
│   │   ├── approval.ts   # 审批接口
│   │   └── ...
│   ├── assets/           # 资源文件 (图片、样式等)
│   ├── components/       # 公共组件
│   ├── layout/           # 布局组件
│   ├── router/           # 路由配置
│   ├── stores/           # 状态管理 (Pinia)
│   ├── types/            # TypeScript 类型定义
│   ├── utils/            # 工具函数
│   │   ├── request.ts    # Axios 请求封装
│   │   └── websocket.ts  # WebSocket 封装
│   ├── views/            # 页面组件
│   │   ├── login/        # 登录页
│   │   ├── home/         # 首页
│   │   ├── todo/         # 待办事项
│   │   ├── approval/     # 审批管理
│   │   └── ...
│   ├── App.vue           # 根组件
│   └── main.ts           # 入口文件
├── .env.development      # 开发环境变量
├── .env.production       # 生产环境变量
├── package.json          # 项目依赖配置
├── tsconfig.json         # TypeScript 配置
├── vite.config.ts        # Vite 构建配置
└── README.md             # 项目说明
```

## 技术栈

- **框架**: Vue 3 (Composition API)
- **语言**: TypeScript
- **UI 库**: Element Plus
- **路由**: Vue Router
- **状态管理**: Pinia（）
- **HTTP 客户端**: Axios
- **构建工具**: Vite
- **代码规范**: ESLint + Prettier


## 前置要求

在开始之前，请确保您的电脑上已安装以下软件：

### 1. Node.js (必需)
- **版本要求**: Node.js 16.x 或更高版本
- **下载地址**: https://nodejs.org/
- **验证安装**:
  ```bash
  node -v
  # 应该显示类似 v16.x.x 或更高版本

  npm -v
  # 应该显示类似 8.x.x 或更高版本
  ```

### 3. 代码编辑器 (推荐)
- **推荐使用**: [VS Code](https://code.visualstudio.com/)
- **推荐插件**:
  - Vue Language Features (Volar)
  - TypeScript Vue Plugin (Volar)
  - ESLint
  - Prettier

### 4. 包管理器 (可选但推荐)
- **推荐使用**: pnpm (比 npm 更快)
- **安装命令**:
  ```bash
  npm install -g pnpm
  ```

## 环境配置

### 1. 检查环境变量文件

项目根目录应该包含以下文件：
- `.env.development` - 开发环境配置
- `.env.production` - 生产环境配置

### 2. 开发环境配置说明

打开 `.env.development` 文件，确认以下配置：

```env
# API 服务地址
VITE_API_BASE_URL=http://127.0.0.1:8888

# WebSocket 服务地址
VITE_WS_BASE_URL=ws://127.0.0.1:9000
```

**重要提示**:
- 这些地址对应后端服务，请确保后端服务已经启动
- 如果后端地址不同，请修改为实际地址

### 3. 后端服务准备

**本项目依赖后端服务才能正常运行**，请确保：
- 后端 API 服务运行在 `http://127.0.0.1:8888`
- WebSocket 服务运行在 `ws://127.0.0.1:9000`

如何启动后端服务，请参考后端项目的文档。

## 启动项目

### 开发模式启动

```bash
# 使用 npm
npm run dev

# 或使用 pnpm
pnpm dev
```

**成功提示**:
```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:3000/
  ➜  Network: http://192.168.x.x:3000/
  ➜  press h + enter to show help
```

### 访问应用

1. 打开浏览器 (推荐使用 Chrome 或 Edge)
2. 访问地址: **http://localhost:3000**
3. 您应该看到登录页面
